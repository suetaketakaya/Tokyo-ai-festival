## 使用方法

### ターミナルQRコード
```bash
./remoteclaude-server --port=8095
# → ターミナルに直接QRコード表示
```

### Webブラウザ
```
http://192.168.11.102:8095/
# → ブラウザでQRコード画像を表示
```

### QRコード画像ファイル
```
qr-code.png
# → 高品質PNG画像として保存
```

## 💡 技術詳細

### QRコードライブラリ:
- **ライブラリ**: `github.com/skip2/go-qrcode`
- **エラー訂正**: Medium (約15%復元可能)
- **サイズ**: 256x256 pixel
- **フォーマット**: PNG

### 表示方式:
1. **ターミナル**: UTF-8文字による視覚表示
2. **画像ファイル**: PNG形式で永続保存
3. **Web**: HTTP経由で画像提供
