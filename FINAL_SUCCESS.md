## **実際の使用フロー**

### **Step 1: サーバー起動**
```bash
./remoteclaude-server --port=8090
# → QRコード表示 + WebSocket待機
```

### **Step 2: iPhone接続**  
```bash
1. Expo QRコードをスキャン → アプリ起動
2. サーバーQRコードをスキャン → WebSocket接続
3. 自動でプロジェクト一覧取得 → 開発画面表示
```

### **Step 3: Claude実行**
```bash
iPhone画面でコマンド入力:
"ls -la" → Claude CLI実行 → リアルタイム結果表示
```
